# System Helpers

This directory contains documentation for the various helper classes and utilities available in the `System/Helpers` namespace.

## Data Manipulation

- [Array](array-helper.md) - Fluent array manipulation and collections.
- [String](string-helper.md) - String manipulation, UUID generation, and inflectors.
- [Number](number-helper.md) - Formatting, conversion, and mathematical utilities.
- [DateTime](datetime-helper.md) - Enhanced date and time handling extending Carbon.

## Web & HTTP

- [Request](request-helper.md) - Handling incoming HTTP inputs and files.
- [Response](response-helper.md) - Generating and sending server responses.
- [Http](http-helper.md) - Cookies, Headers, and General HTTP utilities.
- [Redirect](redirect-helper.md) - Fluent redirection handling.
- [Flash](flash-helper.md) - One-time session messages (Feedback & Errors).
- [Session](session-helper.md) - Persistent session data management.
- [URL](url-helper.md) - URL generation, [Current URL](current-url-helper.md), and [Request URI](request-uri-helper.md).
- [Route](route-helper.md) - Named route resolution and routing assistance.
- [Client & Curl](curl-helper.md) - External HTTP client and Curl wrapper.
- [Html](html-helper.md) - Form builders and HTML generation.
- [Agent](agent-helper.md) - User agent parsing and detection.

## Core Utilities

- [Capsule](capsule-helper.md) - Immutable data container with schema validation.
- [Data](data-helper.md) - Unified data wrapper for collections.
- [DTO](dto-helper.md) - Data Transfer Object base class.
- [Benchmark](benchmark-helper.md) - Execution time and memory benchmarking.
- [Macroable](helpers/macroable.md) - Trait for adding runtime methods to classes.
- [Mixins](helpers/mixins.md) - Trait for bulk-registering macros from a class or object.
- [Lottery](lottery-helper.md) - Probabilistic operations and weighted selection.

## System & Security

- [Encryption](encryption-helper.md) - Data encryption and decryption.
- [File](file-helper.md) - Filesystem operations and directory management.
- [Validation](validation-helper.md) - Input validation and rule management.
- [Format](format-helper.md) - Data formatting utilities.
- [Routine](routine-helper.md) - Common procedural routines.
- [VarDump](vardump-helper.md) - Debugging and variable inspection.
